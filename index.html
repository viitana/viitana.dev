Hi! Welcome to my website. I'm Atte, a backend-focused fullstack developer. I'm passionate about all things backend and games. I work mostly with Golang and C#, with the occasional project in C/C++ or JS/TS/React.

For now, this page simply lists some of my recent personal projects.

Projects

    wikirun.viitana.dev
  • Eternal Gambit - A gamified online chess game with Anime-style 3D graphics and character pieces.

  • Deepest Darkest Depths - A 2.5D roguelike dungeon crawler with a focus on resource management and exploration.
    Steam page coming later™
  • projector - An experimental Vulkan-based asynchronous reprojection demo application.

  • Triangler - A rudimentary OpenGL 3D graphics engine and model viewer built in C++.

    github.com/viitana/runitetimer
  • terminal3D - A standalone-ish little 3D engine that outputs to any ncurses-capable terminal.
    github.com/viitana/terminal3D
  • viitana.dev - This website.
